# README --- Ejercicios de Grid, Flexbox y Animaciones (Resumen)

## 1. Ejercicios de Grid

Los ejercicios de **CSS Grid** muestran cómo crear y organizar
cuadrículas.\
Incluyen ejemplos de columnas y filas con `grid-template-columns` y
`grid-template-rows`, uso de `grid-template-areas` para layouts con
encabezado, barra lateral y pie, y control del espacio con `gap`.\
También se usan `grid-column` y `grid-row` para ampliar el tamaño de
ciertos elementos y se implementa un diseño responsive con
`repeat(auto-fill, minmax())`.

------------------------------------------------------------------------

## 2. Ejercicios de Flexbox

Los ejemplos de **Flexbox** explican cómo alinear y distribuir elementos
dentro de un contenedor flexible.\
Se enseña `justify-content` y `align-items` para centrar, `flex-wrap`
para dividir elementos en varias filas, y `flex-grow` / `flex-basis`
para controlar tamaños proporcionales.\
Incluye casos como barras de navegación, cuadrículas con `flex-wrap`, y
ajustes responsive.

------------------------------------------------------------------------

## 3. Ejercicios de Animación

Los ejercicios de **animaciones CSS** muestran cómo usar `@keyframes` y
`transform` para crear efectos visuales.\
Incluyen ejemplos de *fade-in/fade-out* (opacidad), parpadeo,
desplazamientos con `translate`, rebotes (*bounce*), rotaciones
(`rotate`), zoom con `scale` y combinaciones de color y tamaño.\
Cada animación se repite de forma continua o controlada mediante la
propiedad `animation`.

------------------------------------------------------------------------

Este resumen recopila los principales conceptos de diseño vistos en los
